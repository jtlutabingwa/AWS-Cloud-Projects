## About
 
This repo is a learning journal — each project is something I built from scratch to get hands-on experience with AWS. The goal is to move beyond tutorials and actually understand how these services work together.
 
Each project folder contains its own README with:
- What the project does
- Architecture overview
- Step-by-step instructions
- Key concepts learned
- Cleanup steps
 
## Cost
 
All projects target the **AWS Free Tier** where possible. Any project that may incur costs will call that out clearly. Always follow the cleanup steps to avoid surprise charges.
 
## Prerequisites
 
- An AWS account ([sign up](https://aws.amazon.com/free/))
- AWS CLI installed (optional but recommended)
- Git Bash, WSL, or any Unix-like terminal
- Basic command line familiarity
